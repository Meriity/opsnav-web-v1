# VKlawyers Conveyancing System

This is a modern web-based conveyancing management system designed for law firms to streamline and manage their property transaction workflows. This project aims to improve transparency, document handling, and communication between lawyers and clients throughout the conveyancing process.

## 🚀 Features

- 🧾 Client and Matter Management
- 📂 Document Upload and Archival
- 📊 Dashboard with Key Insights
- 🔍 Search by Matter Number or Client Name
- 🧑‍💼 Role-based Access (Admin & Users)
- 📈 Activity Tracking and Monthly Statistics

## 🛠️ Tech Stack

- **Frontend**: React, TailwindCSS
- **Backend**: Node.js - Express 
- **Database**: MongoDB
